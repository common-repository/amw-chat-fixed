A live chat module that allows members/guests to chat with each other as they browse your site. Includes public chat as well as private messaging.

== Description ==
I fixed it since this seems to be the best free non-third party chat plugin.
-gearsgod

== Installation ==

1. Upload `amw-chat` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
